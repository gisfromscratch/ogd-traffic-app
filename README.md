# ogd-traffic-app
OGD konforme Visualisierung der Straßenverkehrslage. Die Daten werden direkt von dem Endpunkt ["Aktuelle Straßenverkehrslage"](https://opendata.bonn.de/dataset/aktuelle-straßenverkehrslage) der Stadt Bonn beim Starten der Anwendung geladen und in der Karte visualisiert. Die Karte basiert auf der [ArcGIS Runtime .NET](https://developers.arcgis.com/net/latest/).

## Datenquelle:
[Bundesstadt Bonn, Amt 66](https://opendata.bonn.de)
